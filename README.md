# MinimalPay Android

Flat 라이트 테마 + **4페이지(하단 탭)** 구조의 정산 앱입니다.

## 화면 구성 (위저드)

하단 **이전 / 다음** 버튼으로만 이동합니다 (스와이프 비활성).

| 단계 | UC | [다음] 조건 |
|------|-----|-------------|
| 1 | UC-1 | 그룹 생성 + **참여 인원 수**만큼 멤버 전원 등록 |
| 2 | UC-2 | 지출 1건 이상 등록 |
| 3 | UC-3 | 정산 리포트 생성 |
| 4 | UC-4 | 송금 후 [완료] |

## 디자인

- 배경 `#F5F6F8`, 포인트 `#1A73E8`
- Material 카드형 입력·버튼
- 리포트 고정폭(`monospace`) 서식

## 실행 (Android Studio)

1. **File → Open** → `C:\MinimalPayAndroid`
2. Gradle Sync 완료
3. Run ▶

## 시연 순서

1. **그룹** → 생성 → 멤버 2명 (한 명 계좌 생략)
2. **지출** → 등록
3. **정산** → 리포트 생성
4. **송금** → UC-4 Extend 확인
